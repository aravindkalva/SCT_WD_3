# Tic-Tac-Toe Web Application

## Live Demo
https://aravindkalva.github.io/SCT_WD_3/

An interactive and responsive Tic-Tac-Toe game built using HTML, CSS, and JavaScript.

This project allows users to play Tic-Tac-Toe with another player or against the computer while demonstrating JavaScript game logic, DOM manipulation, and event handling.

---

## Features

-  Player vs Player mode
-  Player vs Computer mode
-  Interactive 3×3 game board
-  User click event handling
-  Game state tracking
-  Winner detection
-  Draw detection
-  Restart game functionality
-  Turn indicator display
-  Responsive design

---

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Game Rules

- Player X starts the game.
- Players take turns placing X and O.
- The first player to match three symbols in a row wins.
- If all spaces are filled without a winner, the game ends in a draw.

---

## Implementation

The application uses:

- DOM manipulation for updating the game board
- Event listeners for handling user clicks
- Arrays for storing game state
- Functions for checking winning conditions
- Random computer moves for AI mode
